4.4 Python Packages

Introduction to Python Packages

A Python package is a collection of related modules bundled together. A module is simply a Python file (.py) that contains code like functions, classes, and variables.

Why Use Python Packages?

  • âœ… Organizes Code: Packages help structure large programs neatly.
  • âœ… Reusability: You can use the same package in multiple projects.
  • âœ… Prebuilt Packages: Python provides many standard packages like math, numpy, pandas, etc.
  • âœ… User-Defined Packages: You can create your own custom packages.

Example of a Package in Real Life

Think of a Python package like a folder of tools:

  • Package (Folder Name): science_tools
  • Modules (Inside the Folder): math_tools.py, physics_tools.py, chemistry_tools.py

Writing Python Packages (Creating a User-Defined Package)

To create a package, follow these steps:

  1. Create a new folder, e.g., my_package. This will be your package name.
  2. Inside the my_package folder, create two Python files: math_operations.py and string_operations.py.
  3. Inside my_package, create an empty file named __init__.py. This file tells Python that my_package is a package.

Step 4: Write Code in the Modules

  math_operations.py:
  def add(a, b):
      return a + b
  
  def subtract(a, b):
      return a - b
    
  string_operations.py:
  def reverse_string(s):
      return s[::-1]
  
  def uppercase_string(s):
      return s.upper()

Step 5: Import and Use the Package

  import my_package.math_operations as math_ops
  import my_package.string_operations as str_ops
  
  print(math_ops.add(5, 3))  # Output: 8
  print(str_ops.reverse_string("hello"))  # Output: "olleh"

Using Standard Python Packages

1. math Package (Mathematical Functions)

  import math
  
  print(math.sqrt(25))  # Output: 5.0
  print(math.pi)        # Output: 3.141592653589793
  print(math.factorial(5))  # Output: 120

2. scipy Package (Scientific Computing)

  from scipy import constants
  
  print(constants.g)  # Output: 9.80665 (gravitational acceleration)

3. numpy Package (Numerical Computations)

  import numpy as np
  
  arr = np.array([1, 2, 3, 4])
  print(arr * 2)  # Output: [2 4 6 8]

4. matplotlib Package (Data Visualization)

  import matplotlib.pyplot as plt
  
  x = [1, 2, 3, 4]
  y = [10, 20, 25, 30]
  
  plt.plot(x, y)
  plt.xlabel("X-Axis")
  plt.ylabel("Y-Axis")
  plt.title("Simple Line Graph")
  plt.show()

5. pandas Package (Data Handling)

  import pandas as pd
  
  data = {"Name": ["Alice", "Bob", "Charlie"], "Age": [25, 30, 35]}
  df = pd.DataFrame(data)
  
  print(df)

Using a User-Defined Package

  import my_package.math_operations as math_ops
  
  print(math_ops.add(10, 5))  # Output: 15
