4.2 User-Defined Functions in Python

A function in Python is a block of reusable code that performs a specific task. Instead of writing the same code multiple times, you can create a function and call it whenever needed.

Python provides many built-in functions like print(), len(), and max(), but you can also define your own functions. These are called user-defined functions.

1. Function Definition

A function must be defined before using it.

      
        def function_name(parameters):
            # Function body (code inside function)
            return value  # (Optional)
  • def → Keyword to define a function
  • function_name → Name of the function
  • parameters → Values passed to the function (optional)
  • return → Sends a value back to the caller (optional)

Example of Function Definition:

      
        def greet():
            print("Hello! Welcome to Python.")
        greet()  # Calling the function

2. Function Calling

Once a function is defined, it needs to be called to execute.

      
        def add_numbers():
            a = 5
            b = 10
            print("Sum:", a + b)
        add_numbers()

3. Function Arguments and Parameter Passing

What are Parameters and Arguments?

  • Parameter: A variable listed in the function definition
  • Argument: A value passed when calling a function

Example of Function with Parameters:

      
        def greet(name):
            print("Hello,", name)
        greet("Alice")
        greet("Bob")

Example of Default Arguments:

      
        def greet(name="Guest"):
            print("Hello,", name)
        greet()
        greet("Alice")

Example of Keyword Arguments:

      
        def student_info(name, age):
            print("Name:", name)
            print("Age:", age)
        student_info(age=20, name="John")

Example of Variable-Length Arguments:

      
        def add_numbers(*numbers):
            total = sum(numbers)
            print("Sum:", total)
        add_numbers(5, 10, 15)
        add_numbers(1, 2, 3, 4, 5)

4. Return Statement

A function can return a value using the return statement.

      
        def square(num):
            return num * num
        result = square(4)
        print("Square of 4:", result)

5. Scope of Variables

Local Variables (Defined Inside a Function)

      
        def show_message():
            message = "Hello, Local Variable!"
            print(message)
        show_message()

Global Variables (Defined Outside a Function)

      
        message = "Hello, Global Variable!"
        def show_message():
            print(message)
        show_message()
        print(message)

Modifying Global Variables Inside a Function

      
        count = 10
        def update_count():
            global count
            count += 5
        update_count()
        print("Updated count:", count)

Summary

Feature Description Example
Function Definition Defines a function using def def greet(): print("Hello")
Function Calling Calls a function to execute its code greet()
Arguments & Parameters Passes values to a function greet("Alice")

Conclusion

  • Functions make programs organized, reusable, and efficient.
  • Local variables are used inside functions.
  • Global variables can be used throughout the program.
  • Arguments allow functions to accept input values.
  • The return statement allows functions to send results back.
