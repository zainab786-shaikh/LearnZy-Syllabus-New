4.3 Modules in Python

A module in Python is a file that contains Python code, such as functions, variables, and classes. Modules help organize code into smaller, reusable parts, making programs easier to manage.

For example, instead of writing all the functions in one long script, we can separate them into different files (modules) and use them wherever needed.

1. Writing a Module in Python

To create a module, simply write Python code in a file and save it with a .py extension.

# mymodule.py - This is a Python module
  def greet(name):
      return f"Hello, {name}!"
  
  pi = 3.1416

2. Importing a Module in Python

import mymodule
  
  print(mymodule.greet("Alice"))  # Output: Hello, Alice!
  print(mymodule.pi)              # Output: 3.1416

3. Importing Specific Objects from a Module

from mymodule import greet
  
  print(greet("Bob"))  # Output: Hello, Bob!

4. Using Aliases for Modules and Objects

import mymodule as mm
  
  print(mm.greet("Charlie"))  # Output: Hello, Charlie!

5. Python Built-in Modules

5.1 Numeric and Mathematical Modules

import math
  
  print(math.sqrt(25))  # Output: 5.0
  print(math.factorial(5))  # Output: 120
  print(math.pi)  # Output: 3.141592653589793

Example: Using the random module

import random
  
  print(random.randint(1, 10))
  print(random.choice(["apple", "banana", "cherry"]))

5.2 Functional Programming Module (functools)

from functools import reduce
  
  numbers = [1, 2, 3, 4]
  sum_numbers = reduce(lambda x, y: x + y, numbers)
  
  print(sum_numbers)  # Output: 10

6. Namespace and Scoping in Python

What is a Namespace? A namespace is a space where variable names are stored. Python uses namespaces to keep track of all variables and functions.

Example: Local and Global Scope

x = 10  # Global variable
  
  def my_function():
      y = 5  # Local variable
      print(y)  # Output: 5
  
  print(x)  # Output: 10
  my_function()

Example: Modifying Global Variables inside a Function

count = 0  # Global variable
  
  def increase():
      global count  # Allows modification of the global variable
      count += 1
  
  increase()
  print(count)  # Output: 1

Conclusion

  • Modules help organize Python code into reusable files
  • Importing allows using functions from other files or built-in modules
  • Python built-in modules like math, random, and functools simplify coding
  • Namespace and Scope define where variables exist and how they can be accessed
