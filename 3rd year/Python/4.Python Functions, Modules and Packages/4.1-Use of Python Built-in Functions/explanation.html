4.1 Use of Python Built-in Functions

Python provides several built-in functions that make programming easier. These functions are already included in Python, so you don’t need to install or import anything to use them.

Built-in functions help perform various tasks like:

  • Checking the type of data
  • Converting data from one type to another
  • Performing mathematical calculations
  • Working with strings and numbers

Let’s explore these functions with simple explanations and examples.

1. Type and Data Conversion Functions

These functions help in checking the type of a variable and converting one data type to another.

1.1 type() - Check the Data Type

The type() function tells us what type of data a variable holds.

x = 10
  y = 3.14
  z = "Hello"
  
  print(type(x))  # Output: <class 'int'>
  print(type(y))  # Output: <class 'float'>
  print(type(z))  # Output: <class 'str'>

1.2 Data Type Conversion Functions

Python allows us to convert data from one type to another using the following functions:

  • int() – Converts to an integer
  • float() – Converts to a decimal number
  • str() – Converts to a string
a = "100"
  b = 3.5
  
  num = int(a)
  print(num, type(num))  # Output: 100 <class 'int'>
  
  num2 = int(b)
  print(num2, type(num2))  # Output: 3 <class 'int'>
  
  text = str(num)
  print(text, type(text))  # Output: '100' <class 'str'>

2. Mathematical Functions

2.1 abs() - Absolute Value

The abs() function returns the positive value of a number.

print(abs(-10))  # Output: 10
  print(abs(5))    # Output: 5

2.2 pow() - Power of a Number

print(pow(2, 3))  # Output: 8
  print(pow(5, 2))  # Output: 25

2.3 round() - Round Off a Number

print(round(4.6))   # Output: 5
  print(round(3.2))   # Output: 3
  print(round(3.14159, 2))  # Output: 3.14

2.4 max() and min() - Find Maximum and Minimum

numbers = [10, 20, 5, 8, 25]
  print(max(numbers))  # Output: 25
  print(min(numbers))  # Output: 5

2.5 sum() - Add All Numbers in a List

nums = [1, 2, 3, 4, 5]
  print(sum(nums))  # Output: 15

2.6 math Module Functions

import math
  print(math.sqrt(16))    # Output: 4.0
  print(math.ceil(4.2))   # Output: 5
  print(math.floor(4.9))  # Output: 4

3. String Functions

3.1 len() - Length of a String

text = "Python"
  print(len(text))  # Output: 6

3.2 lower() and upper() - Convert Case

text = "Hello Python"
  print(text.lower())  # Output: hello python
  print(text.upper())  # Output: HELLO PYTHON

3.3 replace() - Replace Text

message = "I love Java"
  print(message.replace("Java", "Python"))  # Output: I love Python

3.4 split() - Split a String

text = "apple,banana,grape"
  fruits = text.split(",")
  print(fruits)  # Output: ['apple', 'banana', 'grape']

4. Input and Output Functions

4.1 input() - Take User Input

name = input("Enter your name: ")
  print("Hello,", name)

4.2 print() - Display Output

print("Welcome to Python!")

Conclusion

Python has many built-in functions that help perform different tasks easily. We explored:

  • Type and data conversion functions (type(), int(), float(), str())
  • Math functions (abs(), pow(), round(), max(), min(), sum(), math.sqrt())
  • String functions (len(), upper(), lower(), replace(), split())
  • Input/output functions (input(), print())

These functions save time and make coding easy.
