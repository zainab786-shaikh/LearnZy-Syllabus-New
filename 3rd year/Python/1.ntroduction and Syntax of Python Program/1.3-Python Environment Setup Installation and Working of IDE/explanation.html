1.3 Python Environment Setup: Installation and Working of IDE

Python is a user-friendly programming language, and setting up the Python environment is simple. In this guide, we will explain step-by-step how to install Python, set up an Integrated Development Environment (IDE), and write your first Python program.

1. What is a Python Environment?

The Python environment is the setup that allows us to write, execute, and test Python programs. It includes:

  • Python Interpreter – The engine that runs Python code.
  • Text Editor/IDE – A software tool for writing and executing Python code.
  • Libraries & Packages – Additional tools for Python programming.

2. How to Install Python?

Step 1: Download Python

  • Go to the official Python website: https://www.python.org
  • Click on "Downloads" and choose your operating system (Windows, macOS, or Linux).
  • Download the latest version (e.g., Python 3.11).

Step 2: Install Python

  • Open the downloaded file and follow the installation steps.
  • Important: Check the box "Add Python to PATH" before clicking "Install Now."
  • After installation, open Command Prompt (Windows) or Terminal (Mac/Linux) and type:
python --version

If Python is installed, it will show the version number (e.g., Python 3.11.2).

3. Python IDEs and Code Editors

An IDE (Integrated Development Environment) is software that makes writing and running Python programs easier.

  • IDLE – Comes with Python (best for beginners).
  • PyCharm – Advanced, feature-rich (for professionals).
  • VS Code – Lightweight and powerful.
  • Jupyter Notebook – Best for data science.
  • Spyder – Used in scientific computing.

4. Writing and Running Python Code in Different IDEs

Method 1: Using IDLE (Comes with Python)

  1. Open IDLE (Installed with Python).
  2. Click on "File" > "New File" and type:
print("Hello, Python!")

Save the file as hello.py and run it by pressing F5 or clicking Run > Run Module.

Method 2: Using PyCharm (Best for Projects)

  1. Download and install PyCharm from https://www.jetbrains.com/pycharm/.
  2. Open PyCharm and create a new project.
  3. Inside the project, create a new Python file and type:
name = input("Enter your name: ") print("Hello, " + name + "!")

Click Run to execute the program.

Method 3: Using VS Code (Best for Beginners)

  1. Download VS Code from https://code.visualstudio.com/.
  2. Install the Python Extension from the VS Code marketplace.
  3. Open a folder, create a Python file (example.py), and type:
for i in range(5): print("Python is fun!")

Click on the Run button or use the terminal:

python example.py

Method 4: Running Python Code in the Terminal

If you don't want to use an IDE, you can write Python code directly in the command line (Windows) or terminal (Mac/Linux).

  1. Open Command Prompt or Terminal.
  2. Type python and press Enter.
  3. Type the following Python code:
print("Hello, Terminal!")

Press Enter and see the output.

5. Setting Up a Virtual Environment (For Advanced Users)

A virtual environment allows you to create a separate Python workspace for each project.

Steps to Create a Virtual Environment:

python -m venv my_project_env

Activate the virtual environment:

Windows:

my_project_env\Scripts\activate

macOS/Linux:

source my_project_env/bin/activate

Now you can install project-specific packages using:

pip install <package_name>

6. Installing and Using Python Libraries

Python has thousands of pre-built libraries to simplify coding.

Example: Installing and Using the requests Library

pip install requests import requests response = requests.get("https://www.python.org") print(response.status_code)

This fetches the Python website’s homepage status.

Conclusion

  • Installing Python is simple and requires only a few steps.
  • Using an IDE like IDLE, PyCharm, or VS Code makes coding easier.
  • Python can be run in a terminal, IDE, or text editor.
  • Virtual environments help manage different projects separately.
