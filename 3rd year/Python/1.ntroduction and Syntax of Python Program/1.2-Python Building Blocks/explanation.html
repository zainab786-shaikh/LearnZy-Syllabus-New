1.2 Python Building Blocks

When writing Python programs, we need to follow some rules and use certain elements correctly. These fundamental elements are called building blocks of Python. They include Identifiers, Keywords, Indentation, Variables, and Comments.

1. Identifiers

Definition: An identifier is the name given to variables, functions, classes, and other objects in Python. It helps the program recognize and use them.

Rules for Identifiers:

  • ‚úÖ Can contain letters (A-Z, a-z), digits (0-9), and underscore (_).
  • ‚úÖ Must not start with a digit (e.g., 1name is invalid).
  • ‚úÖ Cannot use Python keywords as identifiers (e.g., def, class, if).
  • ‚úÖ Case-sensitive (e.g., Name and name are different).

Examples of Valid Identifiers:

      name = "Alice"       # Valid
      age_25 = 25          # Valid
      _employeeID = 101    # Valid

Examples of Invalid Identifiers:

      1name = "Bob"        # Invalid (cannot start with a digit)
      class = "Student"    # Invalid (class is a keyword)
      my-name = "John"     # Invalid (hyphens are not allowed)

2. Keywords

Definition: Keywords are special reserved words in Python that have predefined meanings and cannot be used as variable names.

Keyword Use in Python
if Conditional statement
else Alternative condition
for Loop
while Loop
def Define a function
return Return a value from a function
import Import a module
class Define a class

Example of Keywords in Use:

      if True:
          print("This is a keyword example")

3. Indentation

Definition: Indentation in Python refers to the spaces or tabs at the beginning of a line to define code blocks. Python uses indentation instead of {} (like in C or Java) to group statements.

  • ‚úÖ Python requires indentation (unlike other languages).
  • ‚úÖ Typically, 4 spaces per indentation level is recommended.
  • ‚úÖ Incorrect indentation causes errors.

Example of Proper Indentation:

      if True:
          print("Indented correctly")
          print("Still inside the if-block")
      print("Outside the if-block")

Example of Incorrect Indentation (Error):

      if True:
      print("This will cause an error")  # Incorrect indentation

4. Variables

Definition: A variable is a name that stores data (like numbers, text, lists, etc.). Variables allow us to use and manipulate data in a program.

Rules for Variables:

  • ‚úÖ Can contain letters, numbers, and underscores (_).
  • ‚úÖ Must start with a letter or underscore, not a number.
  • ‚úÖ No special characters (@, $, %) allowed.
  • ‚úÖ Python automatically detects the type of data stored.

Example:

      x = 10        # Integer variable
      name = "Tom"  # String variable
      price = 99.99 # Float variable

Dynamic Typing in Python:

      x = 5       # Integer
      x = "Hello" # Now it's a string
      print(x)    # Output: Hello
