1.5 Python Data Types

Introduction to Python Data Types

Data types in Python define the kind of data a variable can store. For example, a variable can hold numbers, text, or a collection of values. Python provides several built-in data types, and understanding them is essential for programming.

The main data types in Python include:

  • Numbers (int, float, complex)
  • String (str)
  • Tuple (tuple)
  • List (list)
  • Dictionary (dict)
  • Set (set)

1. Numbers (int, float, complex)

Python supports three types of numbers:

  • Integers (int) – Whole numbers (e.g., 10, -5, 200)
  • Floating-point numbers (float) – Numbers with decimal points (e.g., 3.14, -0.99)
  • Complex numbers (complex) – Numbers with real and imaginary parts (e.g., 2+3j) 
      
  # Integer
  x = 10
  y = -25
  print(type(x))  # Output: <class 'int'>
  
  # Float
  a = 3.14
  b = -7.5
  print(type(a))  # Output: <class 'float'>
  
  # Complex Number
  c = 2 + 3j
  d = -4j
  print(type(c))  # Output: <class 'complex'>

2. String (str)

A string is a sequence of characters enclosed in single (' '), double (" "), or triple quotes (''' '''). Strings are used to store text. 

      
  # Single-line strings
  name = "Alice"
  greeting = 'Hello, World!'
  
  # Multi-line string
  message = """This is a 
  multi-line string."""
  print(message)

3. Tuple (tuple)

A tuple is an ordered and immutable (unchangeable) collection of values, written inside parentheses (). 

      
  # Creating a tuple
  fruits = ("apple", "banana", "cherry")
  numbers = (10, 20, 30)

4. List (list)

A list is an ordered, mutable (changeable) collection of values, written inside square brackets []. 

      
  # Creating a list
  colors = ["red", "blue", "green"]
  numbers = [1, 2, 3, 4, 5]

5. Dictionary (dict)

A dictionary is an unordered collection of key-value pairs, written inside curly braces {}. 

      
  # Creating a dictionary
  student = {
      "name": "John",
      "age": 20,
      "marks": 85
  }

6. Set (set)

A set is an unordered collection of unique values, written inside curly braces {}. It does not allow duplicate values. 

      
  # Creating a set
  fruits = {"apple", "banana", "cherry", "apple"}
  print(fruits)  # Output: {'banana', 'apple', 'cherry'}

Conclusion

Python provides a variety of data types to store different kinds of information.

  • Numbers are used for mathematical calculations.
  • Strings store text data.
  • Tuples are immutable collections.
  • Lists are mutable collections.
  • Dictionaries store key-value pairs.
  • Sets store unique values and allow mathematical operations.

By understanding and using these data types correctly, you can write more efficient and organized Python programs.
