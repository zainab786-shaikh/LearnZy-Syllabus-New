6.2 File Handling in Python

File handling is one of the most important features of Python. It allows us to store, retrieve, and manipulate data in files. Instead of losing data when a program stops, we can save it in a file and access it anytime.

Why is File Handling Important?

  • It helps store data permanently.
  • It allows reading and writing large amounts of data efficiently.
  • It is useful for applications like data logging, report generation, and data storage.

1. Opening a File in Different Modes

Before working with a file, we need to open it using Python’s open() function. The open() function allows us to open a file in different modes.

file = open("filename.txt", "mode")

File Modes in Python

ModeDescription
'r'Read mode (default). Opens the file for reading.
'w'Write mode. Creates a new file if it doesn’t exist or overwrites if it exists.
'a'Append mode. Adds new data to the file without deleting existing content.
'x'Exclusive creation mode. Creates a file but gives an error if it already exists.
'b'Binary mode (used with other modes like 'rb' or 'wb').
't'Text mode (default, used with other modes like 'rt' or 'wt').

Examples: Opening Files in Different Modes

# Opening a file in Read Mode ('r')
  file = open("sample.txt", "r")
  print(file.read())
  file.close()
# Opening a file in Write Mode ('w')
  file = open("sample.txt", "w")
  file.write("Hello, this is a new file!")
  file.close()
# Opening a file in Append Mode ('a')
  file = open("sample.txt", "a")
  file.write("\nAppending new content!")
  file.close()

2. Accessing File Contents Using Standard Library Functions

# Using read() Method
  file = open("sample.txt", "r")
  content = file.read()
  print(content)
  file.close()
# Using readline() Method
  file = open("sample.txt", "r")
  line1 = file.readline()
  print(line1)
  file.close()
# Using readlines() Method
  file = open("sample.txt", "r")
  lines = file.readlines()
  print(lines)
  file.close()

3. Writing and Appending to a File

# Writing to a File (write())
  file = open("sample.txt", "w")
  file.write("Python file handling is easy!")
  file.close()
# Appending to a File (write())
  file = open("sample.txt", "a")
  file.write("\nAppending new information.")
  file.close()

4. Closing a File (close())

file = open("sample.txt", "r")
  print(file.read())
  file.close()

5. Renaming and Deleting Files

# Renaming a File
  import os
  os.rename("oldname.txt", "newname.txt")
# Deleting a File
  import os
  os.remove("newname.txt")

6. Directories in Python

# Creating a Directory (mkdir())
  import os
  os.mkdir("NewFolder")
# Removing a Directory (rmdir())
  import os
  os.rmdir("NewFolder")
# Listing Files in a Directory (listdir())
  import os
  print(os.listdir())

7. File and Directory Related Standard Functions

FunctionDescription
open()Opens a file.
read()Reads file content.
write()Writes to a file.
close()Closes the file.
rename(old, new)Renames a file.
remove(file)Deletes a file.
mkdir(dir)Creates a directory.
rmdir(dir)Deletes a directory.
listdir()Lists files and directories.

Conclusion

File handling in Python allows you to create, read, write, rename, delete, and manage files and directories easily. Understanding these concepts is important for working with data storage, logging, and automation.
