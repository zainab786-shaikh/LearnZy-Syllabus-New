6.1 I/O Operations in Python

I/O (Input/Output) operations are used to interact with the user and the computer. In Python, we can:

  • Take input from the keyboard (Reading input)
  • Display output on the screen (Printing output)

Let's understand these concepts in detail with simple explanations and examples.

1. Reading Keyboard Input

Python allows us to take input from the user using the input() function.

Basic Syntax of input()

variable_name = input("Message to user: ")
  • The input() function waits for the user to type something.
  • The text inside input("...") is displayed as a message to guide the user.
  • Whatever the user types is stored in the variable_name as a string.

Example 1: Taking a User's Name as Input

name = input("Enter your name: ")
  print("Hello, " + name + "!")

Output:

Enter your name: John
  Hello, John!

2. Converting Input to a Number

By default, input() always stores data as a string.

To take numbers as input, we need to convert them to an integer (int) or a decimal (float).

Example 2: Adding Two Numbers

num1 = int(input("Enter first number: "))
  num2 = int(input("Enter second number: "))
  
  sum = num1 + num2
  print("The sum is:", sum)

Output:

Enter first number: 5
  Enter second number: 10
  The sum is: 15

3. Taking Multiple Inputs in One Line

Python allows users to enter multiple values at once using split().

Example 3: Taking Name and Age in One Line

name, age = input("Enter your name and age (separated by space): ").split()
  print("Your name is", name, "and you are", age, "years old.")

Output:

Enter your name and age (separated by space): Alice 21
  Your name is Alice and you are 21 years old.

2. Printing Output to the Screen

In Python, we use the print() function to display output on the screen.

Basic Syntax of print()

print("Message to display")

Example 4: Printing Simple Messages

print("Welcome to Python Programming!")
  print("Python is easy to learn.")

Output:

Welcome to Python Programming!
  Python is easy to learn.

Example 5: Printing a Variable

name = "Alice"
  print("Hello,", name)

Output:

Hello, Alice

Example 6: Printing Numbers and Calculations

x = 10
  y = 5
  print("The sum of x and y is:", x + y)

Output:

The sum of x and y is: 15

4. Using sep and end in print()

Example 8: Custom Separator (sep)

print("Apple", "Banana", "Cherry", sep=", ")

Output:

Apple, Banana, Cherry

Example 9: Custom Ending (end)

print("Hello", end=" ")
  print("World!")

Output:

Hello World!

5. Formatting Output using f-strings

Instead of using multiple commas, we can use f-strings for better readability.

Example 10: Using f-strings

name = "Charlie"
  age = 30
  print(f"My name is {name} and I am {age} years old.")

Output:

My name is Charlie and I am 30 years old.
