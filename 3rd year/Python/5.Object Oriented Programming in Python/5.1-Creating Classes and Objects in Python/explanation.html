5.1 Creating Classes and Objects in Python

Introduction

Python is an object-oriented programming (OOP) language. This means it allows us to create objects that represent real-world things like cars, people, or bank accounts.

To create these objects, we need classes. A class is like a blueprint or template, and an object is an actual instance made using that blueprint.

Example in Real Life:

  • Class: A "Car Blueprint" (describes a carâ€™s color, model, speed, etc.)
  • Object: A real car built from that blueprint (e.g., a red Toyota, a blue Honda, etc.)

What is a Class?

A class is a template that defines the properties (variables) and behaviors (functions) of an object.

Syntax for Creating a Class

    
    class ClassName:
        # Class body (variables and methods)

Example of a Class

    
    class Car:
        brand = "Toyota"
        color = "Red"

What is an Object?

An object is an instance of a class. If the class is a blueprint, an object is the actual product created using that blueprint.

Syntax for Creating an Object

    
    object_name = ClassName()

Example of Creating an Object

    
    my_car = Car()
    print(my_car.brand)  # Output: Toyota
    print(my_car.color)  # Output: Red

Using the __init__() Method (Constructor)

In Python, the __init__() method is a special function that automatically runs when we create an object. It is used to initialize attributes.

Example of __init__() Method

    
    class Car:
        def __init__(self, brand, color):
            self.brand = brand
            self.color = color
  
    # Creating objects with different values
    car1 = Car("Toyota", "Red")
    car2 = Car("Honda", "Blue")
  
    print(car1.brand, car1.color)  # Output: Toyota Red
    print(car2.brand, car2.color)  # Output: Honda Blue

Adding Methods (Functions in a Class)

A method is a function defined inside a class that performs an action.

Example: Creating a Method

    
    class Car:
        def __init__(self, brand, color):
            self.brand = brand
            self.color = color
  
        def show_details(self):  # Method
            print(f"This is a {self.color} {self.brand}.")
  
    # Creating objects
    car1 = Car("Toyota", "Red")
    car2 = Car("Honda", "Blue")
  
    # Calling the method
    car1.show_details()  # Output: This is a Red Toyota.
    car2.show_details()  # Output: This is a Blue Honda.

Creating Multiple Objects

Example: Multiple Objects

    
    class Student:
        def __init__(self, name, age):
            self.name = name
            self.age = age
  
        def introduce(self):
            print(f"Hello, my name is {self.name} and I am {self.age} years old.")
  
    # Creating multiple student objects
    student1 = Student("Alice", 20)
    student2 = Student("Bob", 22)
  
    student1.introduce()  # Output: Hello, my name is Alice and I am 20 years old.
    student2.introduce()  # Output: Hello, my name is Bob and I am 22 years old.

Class vs. Object (Key Differences)

Feature Class Object
Definition A blueprint/template A real instance of a class
Example Car class car1 = Car("Toyota", "Red")
Purpose Defines attributes and methods Holds real data

Modifying Object Attributes

    
    car = Car("Ford", "Black")
    print(car.color)  # Output: Black
  
    car.color = "White"  # Changing the color
    print(car.color)  # Output: White

Deleting Objects and Attributes

Example: Deleting an Attribute

    
    del car.color  # Deletes the color attribute

Example: Deleting an Object

    
    del car  # Deletes the entire car object

Real-World Example: Bank Account

    
    class BankAccount:
        def __init__(self, owner, balance):
            self.owner = owner
            self.balance = balance
  
        def deposit(self, amount):
            self.balance += amount
            print(f"{amount} deposited. New balance: {self.balance}")
