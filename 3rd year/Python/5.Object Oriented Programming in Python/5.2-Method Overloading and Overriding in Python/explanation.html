5.2 Method Overloading and Overriding in Python

Method Overloading and Method Overriding are two important concepts in Object-Oriented Programming (OOP). They allow us to define multiple methods in a class with the same name but with different behaviors.

ðŸ”¹ What is Method Overloading?

Method Overloading means creating multiple methods with the same name but different numbers or types of parameters.

ðŸ’¡ Key Point: Python does not support method overloading like other languages (Java, C++). However, we can achieve similar behavior using default arguments or *args (variable arguments).

âœ… Example of Method Overloading Using Default Arguments

class MathOperations:
      def add(self, a, b, c=0):  # c is an optional argument
          return a + b + c
  
  math_obj = MathOperations()
  print(math_obj.add(2, 3))       # Output: 5
  print(math_obj.add(2, 3, 4))    # Output: 9

âœ… Example of Method Overloading Using *args

class MathOperations:
      def add(self, *args):  # *args allows multiple inputs
          return sum(args)
  
  math_obj = MathOperations()
  print(math_obj.add(2, 3))          # Output: 5
  print(math_obj.add(2, 3, 4))       # Output: 9
  print(math_obj.add(1, 2, 3, 4, 5)) # Output: 15

ðŸ”¹ What is Method Overriding?

Method Overriding allows a child class to redefine (modify) a method of its parent class to provide a different behavior.

âœ… Example of Method Overriding

class Animal:
      def sound(self):
          print("Animals make sounds.")
  
  class Dog(Animal):
      def sound(self):  # Overriding the sound method
          print("Dog barks.")
  
  a = Animal()
  d = Dog()
  a.sound()  # Output: Animals make sounds.
  d.sound()  # Output: Dog barks.

âœ… Example of Method Overriding Using super()

class Animal:
      def sound(self):
          print("Animals make sounds.")
  
  class Dog(Animal):
      def sound(self):
          super().sound()  # Calls the parent class method
          print("Dog barks.")
  
  d = Dog()
  d.sound()

ðŸ”¹ Difference Between Method Overloading and Method Overriding

Feature Method Overloading Method Overriding
Definition Multiple methods with the same name but different parameters. A child class redefines a method from the parent class.
Support in Python Not directly supported. Achieved using default arguments or *args. Fully supported in Python.
Number of Classes Happens in the same class. Happens in different classes (inheritance).
Purpose Provides multiple ways to use the same method. Changes the behavior of an inherited method.

ðŸ”¹ Real-Life Example: Overloading vs Overriding

Scenario 1: Method Overloading (Same Class)

class Calculator:
      def add(self, a, b, c=0):
          return a + b + c
  
  calc = Calculator()
  print(calc.add(5, 10))     # Output: 15
  print(calc.add(5, 10, 20)) # Output: 35

Scenario 2: Method Overriding (Parent and Child Classes)

class Vehicle:
      def fuel(self):
          print("Vehicles use fuel.")
  
  class Car(Vehicle):
      def fuel(self):
          print("Cars use petrol or diesel.")
  
  class ElectricCar(Vehicle):
      def fuel(self):
          print("Electric cars use batteries.")
  
  v = Vehicle()
  c = Car()
  e = ElectricCar()
  v.fuel()  # Output: Vehicles use fuel.
  c.fuel()  # Output: Cars use petrol or diesel.
  e.fuel()  # Output: Electric cars use batteries.
