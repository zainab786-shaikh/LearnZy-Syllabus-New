2.2 Control Flow

Control flow in Python refers to the order in which statements and instructions are executed in a program. By default, Python runs code line by line from top to bottom, but control flow statements allow us to make decisions, repeat actions, and jump to specific parts of the program based on conditions.

Types of Control Flow in Python

Python provides three main types of control flow statements:

  • Conditional Statements (Decision Making) – Used to make choices in a program (e.g., if, if-else, if-elif-else).
  • Loops (Iteration Statements) – Used to repeat a block of code multiple times (e.g., for loop, while loop).
  • Jump Statements – Used to control the flow inside loops (e.g., break, continue, pass).

Why is Control Flow Important?

Control flow allows Python programs to:

  • ✅ Make decisions (e.g., Check if a user is eligible to vote).
  • ✅ Repeat tasks efficiently (e.g., Print numbers from 1 to 10).
  • ✅ Skip or exit loops when needed (e.g., Stop searching when a value is found).
