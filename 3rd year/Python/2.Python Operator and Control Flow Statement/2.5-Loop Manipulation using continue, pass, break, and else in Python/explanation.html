2.5 Loop Manipulation using continue, pass, break, and else in Python

Loops in Python help us repeat tasks without writing the same code multiple times. But sometimes, we may want to skip some steps, stop the loop early, or do something when the loop finishes completely. This is where continue, pass, break, and else come into play.

1. break Statement (Stops the Loop Immediately)

The break statement is used to stop a loop completely when a certain condition is met.

      
  for num in range(1, 10):  # Loop from 1 to 9
      if num == 5:
          print("Number 5 found! Stopping loop.")
          break  # Stop the loop when num is 5
      print("Current number:", num)

2. continue Statement (Skip to the Next Iteration)

The continue statement skips the current iteration and moves to the next one.

      
  for num in range(1, 10):  
      if num == 5:
          print("Skipping number 5")
          continue  # Skip the rest of the code and move to the next number
      print("Current number:", num)

3. pass Statement (Do Nothing, Just a Placeholder)

The pass statement is used when a statement is required syntactically, but we donâ€™t want to do anything.

      
  for num in range(1, 6):  
      if num == 3:
          pass  # Do nothing when num is 3
      print("Current number:", num)

4. else Clause in Loops

The else statement in loops runs only when the loop completes normally (i.e., it is not stopped by break).

      
  for num in range(1, 6):
      print("Current number:", num)
  else:
      print("Loop finished without a break.")

Summary Table

Statement What It Does
break Stops the loop completely
continue Skips the current iteration and moves to the next one
pass Does nothing (just a placeholder)
else (in loop) Runs only if the loop completes without break

Real-Life Example: Password Checking

We allow 3 attempts for password entry. If correct, we stop using break. If all attempts fail, we use else.

      
  correct_password = "python123"
  
  for attempt in range(3):  # Allow 3 attempts
      password = input("Enter password: ")
      if password == correct_password:
          print("Access granted!")
          break  # Stop the loop if password is correct
      print("Incorrect password. Try again.")
  else:
      print("Too many incorrect attempts. Access denied!")

Conclusion

  • break stops the loop.
  • continue skips the current iteration and moves to the next.
  • pass does nothing (used as a placeholder).
  • else runs only if the loop completes without break.
