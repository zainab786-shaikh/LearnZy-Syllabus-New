2.4 Looping in Python

Loops are used in Python to execute a block of code multiple times without writing the same code again and again. Instead of writing repetitive instructions, loops allow us to repeat actions efficiently.

Python provides different types of loops:

  • while loop
  • for loop
  • Nested loops (A loop inside another loop)

1. while Loop in Python

The while loop executes a block of code as long as the condition is true. When the condition becomes false, the loop stops.

      
  while condition:
      # Code to execute

Example 1: Printing numbers from 1 to 5

      
  num = 1
  while num <= 5:
      print(num)
      num += 1  # Increases num by 1 in each loop

2. for Loop in Python

A for loop is used to iterate over a sequence (like a list, string, or range of numbers).

      
  for variable in sequence:
      # Code to execute

Example 1: Printing numbers from 1 to 5

      
  for num in range(1, 6):
      print(num)

3. Nested Loops in Python

A nested loop is a loop inside another loop.

Example: Printing a Pattern

      
  for i in range(1, 4):   # Outer loop (rows)
      for j in range(1, 4):  # Inner loop (columns)
          print("*", end=" ")
      print()  # Moves to the next line

When to Use Which Loop?

Loop Type Best For
while loop When the number of iterations is unknown (repeat until a condition is false)
for loop When looping through a sequence (like lists, strings, or a range of numbers)
Nested loops When working with tables, patterns, or multi-level structures

Conclusion

  • ✅ while loop runs until a condition is false.
  • ✅ for loop runs for a fixed number of times.
  • ✅ Nested loops allow looping inside another loop.
