3.3 Sets in Python

A set in Python is an unordered collection of unique items. This means that no element can appear more than once, and the items do not have a specific order. Sets are commonly used when you need to store unique values and perform mathematical set operations like union, intersection, and difference.

a) Accessing Values in a Set, Deleting Values, and Updating Sets

Creating a Set

      
        # Creating a set
        my_set = {1, 2, 3, 4, 5}
        print(my_set)  # Output: {1, 2, 3, 4, 5}
  
        # Using set() function
        another_set = set([10, 20, 30])
        print(another_set)  # Output: {10, 20, 30}

Accessing Values in a Set

      
        my_set = {"apple", "banana", "cherry"}
        for item in my_set:
            print(item)

Adding Items to a Set

      
        fruits = {"apple", "banana"}
        fruits.add("cherry")
        fruits.update(["orange", "grape"])

Deleting Values from a Set

      
        my_set = {10, 20, 30, 40, 50}
        my_set.remove(20)
        my_set.discard(100)
        popped_item = my_set.pop()
        my_set.clear()
        del my_set

b) Basic Set Operations

Union

      
        set_a = {1, 2, 3}
        set_b = {3, 4, 5}
        print(set_a | set_b)  # Output: {1, 2, 3, 4, 5}

Intersection

      
        print(set_a & set_b)  # Output: {3}

Difference

      
        print(set_a - set_b)  # Output: {1, 2}

Symmetric Difference

      
        print(set_a ^ set_b)  # Output: {1, 2, 4, 5}

c) Built-in Set Functions

len(), max(), min(), sum()

      
        numbers = {10, 25, 35, 5, 15}
        print(len(numbers))
        print(max(numbers))
        print(min(numbers))
        print(sum(numbers))

sorted(), any(), all()

      
        my_set = {3, 1, 4, 5, 2}
        print(sorted(my_set))  # Output: [1, 2, 3, 4, 5]

copy()

      
        original_set = {1, 2, 3}
        copied_set = original_set.copy()
        print(copied_set)  # Output: {1, 2, 3}
