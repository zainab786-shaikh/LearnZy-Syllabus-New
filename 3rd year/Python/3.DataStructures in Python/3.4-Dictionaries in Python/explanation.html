3.4 Dictionaries in Python

A dictionary in Python is a collection of key-value pairs. It is used to store data in an organized way, where each key has a corresponding value.

What is a Dictionary?

A dictionary is written using curly brackets {}, and the key-value pairs are separated by colons :

      
  # Creating a dictionary
  student = {
      "name": "John",
      "age": 20,
      "grade": "A"
  }
  print(student)
  # Output: {'name': 'John', 'age': 20, 'grade': 'A'}

(a) Accessing, Deleting, and Updating Dictionary Values

1. Accessing Values in a Dictionary

      
  print(student["name"])  # Output: John
  print(student["age"])   # Output: 20
  print(student.get("grade"))  # Output: A
  print(student.get("city", "Not Found"))  # Output: Not Found

2. Updating a Dictionary

      
  student["age"] = 21
  student["city"] = "New York"
  print(student)
  # Output: {'name': 'John', 'age': 21, 'grade': 'A', 'city': 'New York'}

3. Deleting Values from a Dictionary

      
  del student["city"]
  print(student)
  # Output: {'name': 'John', 'age': 21, 'grade': 'A'}

(b) Basic Dictionary Operations

1. Finding the Length of a Dictionary

      
  print(len(student))  # Output: 3

2. Checking if a Key Exists

      
  if "age" in student:
      print("Key exists!")  # Output: Key exists!

3. Looping Through a Dictionary

      
  for key, value in student.items():
      print(f"{key}: {value}")
  # Output:
  # name: John
  # age: 21
  # grade: A

(c) Built-in Dictionary Functions

1. keys() - Get All Keys

      
  print(student.keys())  # Output: dict_keys(['name', 'age', 'grade'])

2. values() - Get All Values

      
  print(student.values())  # Output: dict_values(['John', 21, 'A'])

3. items() - Get All Key-Value Pairs

      
  print(student.items())  # Output: dict_items([('name', 'John'), ('age', 21), ('grade', 'A')])

4. update() - Merge Dictionaries

      
  new_info = {"city": "New York", "phone": "1234567890"}
  student.update(new_info)
  print(student)

5. copy() - Copy a Dictionary

      
  copy_dict = student.copy()
  print(copy_dict)

6. popitem() - Remove the Last Item

      
  student.popitem()
  print(student)

Conclusion

Dictionaries are an essential part of Python and provide a fast, flexible way to store and retrieve data. You can use them for various tasks such as storing user details, keeping track of products in a store, and more.
