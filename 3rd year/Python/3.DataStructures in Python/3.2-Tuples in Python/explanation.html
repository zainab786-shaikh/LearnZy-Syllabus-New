3.2 Tuples in Python

A tuple is a data structure in Python that allows you to store multiple values in a single variable. Tuples are similar to lists but have one key difference: tuples are immutable, meaning that their values cannot be changed after creation.

Example of a Tuple:

fruits = ("apple", "banana", "cherry")
  print(fruits)  
  # Output: ('apple', 'banana', 'cherry')

Here, fruits is a tuple containing three elements.

(a) Accessing, Deleting, and Updating Tuples

1. Accessing Values in Tuples

You can access elements in a tuple using indexing, just like lists. Indexing starts from 0 (zero-based indexing).

colors = ("red", "green", "blue", "yellow")
  print(colors[0])  # Output: red
  print(colors[2])  # Output: blue
  print(colors[-1]) # Output: yellow

Accessing a Range of Elements (Slicing):

numbers = (10, 20, 30, 40, 50)
  print(numbers[1:4])  # Output: (20, 30, 40)

2. Deleting Values in Tuples

Since tuples are immutable, you cannot delete individual elements. However, you can delete the entire tuple.

fruits = ("apple", "banana", "cherry")
  del fruits

3. Updating Tuples

Since tuples cannot be changed, you must create a new tuple by concatenating existing tuples.

tuple1 = (1, 2, 3)
  tuple2 = (4, 5, 6)
  new_tuple = tuple1 + tuple2
  print(new_tuple)  # Output: (1, 2, 3, 4, 5, 6)

(b) Basic Tuple Operations

1. Concatenation (Joining Tuples)

tuple1 = (1, 2, 3)
  tuple2 = (4, 5, 6)
  result = tuple1 + tuple2
  print(result)  # Output: (1, 2, 3, 4, 5, 6)

2. Repetition (Multiplying Tuples)

numbers = (7, 8)
  result = numbers * 3
  print(result)  # Output: (7, 8, 7, 8, 7, 8)

3. Checking Membership

fruits = ("apple", "banana", "cherry")
  print("banana" in fruits)  # Output: True

4. Finding the Length of a Tuple

colors = ("red", "green", "blue")
  print(len(colors))  # Output: 3

(c) Built-in Tuple Functions

1. len() – Returns the Length of a Tuple

numbers = (10, 20, 30, 40)
  print(len(numbers))  # Output: 4

2. max() – Returns the Maximum Value

values = (5, 10, 15, 20)
  print(max(values))  # Output: 20

3. min() – Returns the Minimum Value

values = (5, 10, 15, 20)
  print(min(values))  # Output: 5

4. sum() – Returns the Sum of Elements

numbers = (1, 2, 3, 4)
  print(sum(numbers))  # Output: 10

5. tuple() – Converts a List into a Tuple

my_list = [1, 2, 3]
  my_tuple = tuple(my_list)
  print(my_tuple)  # Output: (1, 2, 3)

6. count() – Counts the Occurrences of an Element

numbers = (1, 2, 2, 3, 4, 2)
  print(numbers.count(2))  # Output: 3

7. index() – Returns the Index of an Element

fruits = ("apple", "banana", "cherry", "banana")
  print(fruits.index("banana"))  # Output: 1

Conclusion

  • ✔ Tuples are immutable, meaning values cannot be changed after creation.
  • ✔ You can access elements using indexing and slicing.
  • ✔ Tuples cannot be updated or deleted, but you can create a new tuple.
  • ✔ Python provides various tuple operations like concatenation, repetition, and membership checking.
  • ✔ There are several built-in functions like len(), max(), min(), sum(), count(), and index() to perform operations on tuples.

Tuples are widely used when you need unchangeable collections of data, such as coordinates, database records, or constant values.
