3.1 Lists in Python

Introduction to Lists

A list in Python is a collection of multiple values stored together in a single variable. Lists allow us to store different types of data such as numbers, strings, or even other lists. They are ordered, mutable (changeable), and allow duplicate values.

a) Defining Lists, Accessing Values, Deleting Values, and Updating Lists

Defining a List

A list is created by enclosing values inside square brackets [ ], separated by commas.

      
        # List of numbers
        numbers = [10, 20, 30, 40, 50]
        
        # List of strings
        fruits = ["Apple", "Banana", "Cherry"]
        
        # Mixed data types
        mixed = [25, "Hello", 3.14, True]
        
        # Empty List
        empty_list = []

Accessing Values in a List

Each item in a list has an index (position number) that starts from 0.

      
        fruits = ["Apple", "Banana", "Cherry"]
        
        print(fruits[0])  # Output: Apple
        print(fruits[1])  # Output: Banana
        print(fruits[2])  # Output: Cherry

Updating a List

      
        numbers = [10, 20, 30, 40]
        numbers[1] = 99  # Changing value at index 1
        print(numbers)   # Output: [10, 99, 30, 40]

Deleting Values from a List

      
        fruits = ["Apple", "Banana", "Cherry"]
        del fruits[1]  # Deletes "Banana"
        print(fruits)   # Output: ["Apple", "Cherry"]

b) Basic List Operations

Concatenation (Joining Lists)

      
        list1 = [1, 2, 3]
        list2 = [4, 5, 6]
        combined = list1 + list2
        print(combined)  # Output: [1, 2, 3, 4, 5, 6]

Finding the Length of a List

      
        numbers = [10, 20, 30, 40, 50]
        print(len(numbers))  # Output: 5

c) Built-in List Functions

Appending an Item

      
        fruits = ["Apple", "Banana"]
        fruits.append("Mango")
        print(fruits)  # Output: ['Apple', 'Banana', 'Mango']

Sorting a List

      
        numbers = [5, 2, 9, 1, 7]
        numbers.sort()
        print(numbers)  # Output: [1, 2, 5, 7, 9]

Conclusion

Python lists are powerful, flexible, and easy to use. You can store multiple values, modify them, and perform various operations with built-in functions. Lists are widely used in programming for managing data efficiently.
