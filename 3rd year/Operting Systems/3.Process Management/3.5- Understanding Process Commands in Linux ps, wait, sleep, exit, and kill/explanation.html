3.5 Understanding Process Commands in Linux: ps, wait, sleep, exit, and kill

Processes are programs that are currently running on a computer. In Linux, we use different commands to manage these processes, such as viewing them, stopping them, or making them wait.

In this guide, we will explore five important process-related commands:

  • ps – Shows the list of running processes.
  • wait – Waits for a process to complete.
  • sleep – Pauses execution for a specified time.
  • exit – Ends a process.
  • kill – Terminates a process forcefully.

1. ps Command – View Running Processes

The ps (process status) command is used to see which processes are currently running on the system.

Example 1: Display all processes for the current user

ps

Output:

PID TTY          TIME CMD
  1234 pts/0    00:00:00 bash
  5678 pts/0    00:00:00 ps

2. wait Command – Wait for a Process to Complete

The wait command makes the script pause until a process finishes executing.

Example:

#!/bin/bash
  echo "Starting process..."
  sleep 5 &   # Run sleep command in the background
  pid=$!      # Capture process ID
  echo "Waiting for process $pid to complete..."
  wait $pid   # Wait for process to finish
  echo "Process $pid completed!"

3. sleep Command – Pause Execution for a Specified Time

The sleep command delays execution for a given time.

Example:

echo "Waiting for 3 seconds..."
  sleep 3
  echo "Done!"

4. exit Command – End a Process

The exit command is used to terminate a script or process.

Example:

#!/bin/bash
  echo "This script will exit now."
  exit 0

5. kill Command – Terminate a Process

The kill command stops a running process using its Process ID (PID).

Example:

ps aux | grep firefox

Output:

user      3456  5.0  2.1 21580 12340 ?       S    12:15   0:30 firefox

Now, we can kill the process using:

kill 3456

To force kill a process:

kill -9 3456

To kill all instances of a program:

pkill firefox

Summary of Commands

Command Function
ps Displays running processes
wait Waits for a process to finish
sleep Pauses execution for a specified time
exit Ends a process or script
kill Stops a process using its PID
