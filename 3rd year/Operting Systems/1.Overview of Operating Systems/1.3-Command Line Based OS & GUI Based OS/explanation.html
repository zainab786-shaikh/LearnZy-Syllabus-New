2. GUI Based Operating Systems

A Graphical User Interface (GUI) OS allows users to interact using icons, windows, and a mouse instead of typing commands.

2.1 Windows

  • Developed by Microsoft in 1985.
  • One of the most popular OS for personal computers.
  • Uses icons, start menu, taskbar, and file explorer.
  • Very easy to use compared to CLI-based OS like DOS or UNIX.

Conclusion

CLI-based OS (DOS, UNIX) is fast but requires users to remember commands.

GUI-based OS (Windows, Linux) is easier to use with a graphical interface.

Windows is more user-friendly, while Linux is used for security and development.
