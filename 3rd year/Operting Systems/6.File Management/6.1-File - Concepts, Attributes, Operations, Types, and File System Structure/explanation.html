6.1 File - Concepts, Attributes, Operations, Types, and File System Structure

1. What is a File? (Concepts)

A file is a collection of data stored on a computer. It can be anything—text, images, videos, programs, or any digital content.

Think of a file as a book in a library:

  • A book has a title (file name).
  • It contains pages (data).
  • It has an author and a publishing date (file attributes).
  • You can read, write, or edit it (file operations).

Example of Files:

  • Text File (.txt) – A simple text document.
  • Image File (.jpg, .png) – A photo or picture.
  • Video File (.mp4, .avi) – A movie or clip.
  • Executable File (.exe) – A software program.

2. File Attributes

Each file has specific properties known as file attributes. These attributes store metadata (information about the file).

Attribute Description Example
Name The file's name mydocument.txt
Type The format of the file .pdf, .mp3
Size The storage space the file occupies 5MB
Location The file’s path on the system C:\Users\Documents\file.txt
Creation Date When the file was created 22-Feb-2025
Modification Date Last time the file was changed 23-Feb-2025
Permissions Who can read, write, or execute the file Read-Only, Editable
Owner The user who created or owns the file Admin

3. File Operations

A computer system allows users to perform several operations on files.

Operation Description Example
Create Making a new file Creating notes.txt in Notepad
Open Accessing an existing file Opening photo.jpg in Paint
Read Viewing file contents Reading report.pdf in a viewer
Write Modifying file contents Editing resume.docx in MS Word
Delete Removing a file Deleting old_song.mp3
Rename Changing file name Renaming doc1.txt to report.txt
Copy Duplicating a file Copying video.mp4 to a USB drive
Move Transferring file location Moving image.png from Downloads to Desktop

4. Types of Files

(A) Text Files

  • .txt - Plain text file (e.g., notes.txt)
  • .docx - MS Word document (e.g., resume.docx)
  • .html - Webpage file (e.g., index.html)
  • .csv - Spreadsheet text file (e.g., data.csv)

(B) Binary Files

  • .jpg, .png - Image files (e.g., photo.jpg)
  • .mp3, .wav - Audio files (e.g., song.mp3)
  • .mp4, .avi - Video files (e.g., movie.mp4)
  • .exe - Executable program (e.g., chrome.exe)

5. File System Structure

A file system organizes and manages files on a computer.

  • Directories (Folders): Used to group files together.
  • Files: The actual data stored.
  • Paths: Location of a file (e.g., C:\Users\Documents\file.txt).
  • Storage Media: Hard drives, USB drives, SSDs.
  • File Allocation Table (FAT): Keeps track of file locations.

Example of a File System Structure in Windows:

    C:\
    ├── Documents
    │   ├── report.docx
    │   ├── budget.xlsx
    │
    ├── Pictures
    │   ├── vacation.jpg
    │   ├── profile.png
    │
    ├── Music
    │   ├── song.mp3
    │
    ├── Videos
    │   ├── movie.mp4

Conclusion

Understanding files and file systems helps in managing data efficiently on a computer.
