2.4 Use of Operating System Tools

The Operating System (OS) is like the manager of your computer. It helps control different things like users, security, devices, and performance. To make things easier, the OS provides built-in tools that help users manage their computers effectively.

In this topic, we will discuss five important OS tools:

  • User Management
  • Security Policy
  • Device Management
  • Performance Monitor
  • Task Scheduler

Each of these tools plays a key role in keeping the computer system running smoothly. Let's go through them one by one with simple explanations and examples.

1. User Management

User Management is the process of creating, modifying, and deleting user accounts on a computer or network. This tool helps control who can access the system and what they can do.

Why is User Management Important?

  • Prevents unauthorized access to the system.
  • Helps in assigning different permissions to different users.
  • Allows multiple users to use the same computer with their own settings.

Examples of User Management:

  • In Windows OS, you can go to Control Panel → User Accounts to manage users.
  • In Linux, you can use commands like:
    • adduser username → To create a new user.
    • deluser username → To remove a user.

Real-Life Example:

Imagine a school computer lab. The teacher has an admin account that can install software, but students have limited accounts that cannot change system settings. This ensures security and control over the system.

2. Security Policy

A Security Policy in an OS defines rules and settings that protect the computer from threats like hacking, viruses, and unauthorized access.

Why is Security Policy Important?

  • Protects sensitive data from hackers and malware.
  • Prevents unauthorized users from accessing the system.
  • Helps in password management and account security.

Examples of Security Policy:

  • Password Policies:
    • Enforcing strong passwords (at least 8 characters, including symbols).
    • Locking accounts after multiple wrong password attempts.
  • Firewall Settings: Blocks harmful connections from the internet.
  • Antivirus & Windows Defender: Scans the system for viruses and malware.

Real-Life Example:

When you try to log in to your email, and after 3 wrong attempts, your account gets locked for security reasons. This is an example of a security policy in action.

3. Device Management

Device Management refers to the way the OS handles hardware devices like printers, keyboards, and USB drives.

Why is Device Management Important?

  • Helps in installing drivers so that devices work properly.
  • Ensures that devices communicate correctly with the OS.
  • Allows enabling or disabling devices when needed.

Examples of Device Management:

  • In Windows, you can go to Device Manager (devmgmt.msc) to see all connected devices.
  • In Linux, commands like lsusb and lspci list connected devices.
  • If a printer is not working, you can update its driver through Device Manager.

Real-Life Example:

When you connect a new printer to your computer, the OS automatically installs a driver so that you can print documents. If there’s an issue, you might need to update or reinstall the driver using Device Manager.

4. Performance Monitor

A Performance Monitor is a tool that helps check how well your computer is working. It shows details about CPU usage, memory usage, disk activity, and network activity.

Why is Performance Monitor Important?

  • Helps in finding slow or malfunctioning applications.
  • Identifies high CPU or memory usage that may slow down the system.
  • Monitors network activity to detect potential security issues.

Examples of Performance Monitor:

  • Windows Task Manager (Ctrl + Shift + Esc): Shows CPU, RAM, and Disk usage in real-time.
  • Linux: Using top or htop commands: Displays CPU and memory usage in real-time.

Real-Life Example:

If your laptop is running slow, you can open Task Manager and see if a particular program is using too much CPU or RAM. You can then close that program to speed up your system.

5. Task Scheduler

Task Scheduler is a tool that helps automate tasks by scheduling them to run at a specific time.

Why is Task Scheduler Important?

  • Helps in automating system tasks (e.g., backups, updates).
  • Reduces the need for manual work.
  • Ensures that important tasks run even if the user forgets.

Examples of Task Scheduler:

  • Windows Task Scheduler (taskschd.msc): Automatically runs antivirus scans every night.
  • Linux Cron Jobs (crontab -e): Automates tasks like file backups or system updates.

Real-Life Example:

Imagine you want your computer to back up files every night at 2 AM. Instead of doing it manually, you can use Task Scheduler to set up an automatic backup.
