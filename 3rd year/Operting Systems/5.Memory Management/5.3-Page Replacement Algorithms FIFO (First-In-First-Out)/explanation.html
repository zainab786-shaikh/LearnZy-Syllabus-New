5.3 Page Replacement Algorithms: FIFO (First-In-First-Out)

Introduction to Page Replacement

When a computer runs multiple programs, it needs to manage memory efficiently. A program's data and instructions are stored in RAM (main memory), but RAM is limited. If RAM is full and a program needs more space, the operating system (OS) removes some data to make room for new data. This process is called page replacement.

One common page replacement method is FIFO (First-In-First-Out).

FIFO (First-In-First-Out) Page Replacement Algorithm

The FIFO algorithm is simple and works just like a queue. The first page that enters the memory is the first to be removed when a new page is needed.

How FIFO Works (Step-by-Step Explanation)

  • A fixed number of page slots (frames) are available in RAM.
  • Example: Suppose we have 3 frames available in RAM.
  • When a page is needed, it is loaded into an empty frame.
  • If all frames are full and a new page needs to be loaded, the oldest page (first inserted page) is removed to make space for the new page.
  • This process continues, ensuring that the oldest page is always removed first.

Example 2: Real-Life Analogy of FIFO

Imagine a queue at a movie theater. The first person to enter the queue is the first to buy a ticket and leave. New people join at the end of the queue, and when the counter serves a person, they leave. Similarly, in FIFO, the first page that enters memory is the first to be replaced.

Advantages of FIFO

  • ✅ Simple and easy to implement – Works like a queue.
  • ✅ No complex calculations required – Just track the order of arrival.

Disadvantages of FIFO

  • ❌ May remove important pages too early – A page that is needed frequently might be removed.
  • ❌ Not always optimal – Sometimes, replacing an old page is not the best choice.

Conclusion

FIFO is a basic page replacement algorithm that removes the oldest page when memory is full. It is easy to understand and implement but can lead to unnecessary page faults if an important page is removed too soon.
